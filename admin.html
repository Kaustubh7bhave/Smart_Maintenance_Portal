Smart Maintenance Portal

  1. Add Department: The admin can add new departments to the system.
  2. View All Departments: The admin can view all departments in the system and also edit or delete them if required.
  3. View All Data: The admin can view all data in the system, including complaints and other information.
  4. Add Road Maintenance Deadline: The admin can set deadlines for road maintenance activities, and also specify them day-wise.
  5. Add Helpline Numbers: The admin can add contact numbers for helplines or emergency services that may be required by users.
